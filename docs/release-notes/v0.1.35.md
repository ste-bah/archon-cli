# v0.1.35 — Archon Skills Pack + Project Installer

Phase 3 of 3 — 5 archon-specific skills, project init script, and comprehensive docs.

## What's new

- **`/spec-to-tasks`** — Refine the `/prd-to-spec` task tree into atomic, dev-flow-ready task files with verification checklists. Splits/merges tasks, verifies atomicity (single-responsibility, testable, < 1 day, no implicit deps), updates INDEX.md.
- **`/compose-pipeline`** — Sequential chaining of `/to-prd` → `/prd-to-spec` → `/spec-to-tasks` via Skill tool, then hands off for manual `/archon-code`. Confirms with user before starting.
- **`/ci-gate-walker`** — Run `scripts/ci-gate.sh`, surface findings inline, suggest `/diagnose` for failing gates. Does NOT auto-fix.
- **`/setup-archon-skills`** — Interactive 8-prompt first-run wizard covering auth, personality, theme, permissions, embeddings, model+effort, output paths, and project init.
- **`/write-a-skill`** — Meta-skill for authoring new SKILL.md skills. Adapted from mattpocock/skills (MIT).

## Project Installer

`scripts/archon-init.sh` — POSIX-compatible, idempotent project initialiser:

```bash
archon-init.sh [--target <dir>] [--archon-cli-repo <path>] [--no-agents]
```

Creates `.archon/skills/`, `.archon/templates/`, `.archon/adr/`, `.archon/context/`, `prds/`, `tasks/`, and wires `.gitignore`. Safe to re-run.
