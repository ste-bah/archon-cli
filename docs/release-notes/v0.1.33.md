# v0.1.33 — Skills Foundation

Phase 1 of the skill ecosystem inspired by [mattpocock/skills](https://github.com/mattpocock/skills).

## What's new

- **SKILL.md loader extended** — flat-file `.archon/skills/<name>.md` now works alongside subdir `<name>/SKILL.md`. Subdir wins on collision.
- **`/to-prd`** — new skill that injects the ai-agent-prd framework into the conversation. The agent writes `prds/<slug>/PRD.md`. Aliases: `/prd`.
- **`/prd-to-spec`** — new skill that injects the prdtospec framework. The agent writes `tasks/phase<N>/task<M>.md` per phase discovered in the PRD. Aliases: `/decompose-prd`.
- **Templates embedded** — `ai-agent-prd.md` (92KB) and `prdtospec.md` (43KB) are now compiled into the binary via `include_str!()`. Override per-user (`~/.config/archon/templates/`) or per-project (`assets/templates/`).

## Prompt-cache strategy

Phase 1 relies on Anthropic's automatic prefix caching (5-minute TTL). Both templates exceed the 1024-token threshold, so repeat invocations within the cache window hit automatically. No `archon-llm` refactoring required.

## Tests

26 new tests (8 loader, 5 template resolution, 5 /to-prd, 4 /prd-to-spec, 1 registry integration, 3 build checks).
