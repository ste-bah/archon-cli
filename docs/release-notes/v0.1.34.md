# v0.1.34 — Engineering Skills Pack

Phase 2 of 3 — 5 engineering skills adapted from [mattpocock/skills](https://github.com/mattpocock/skills) (MIT).

## What's new

- **`/grill-me`** — non-code alignment session. Relentlessly interviews you about a plan until shared understanding is reached.
- **`/grill-with-docs`** — alignment + documentation. Builds `.archon/context.md` glossary and `.archon/adr/` as decisions crystallise.
- **`/diagnose`** — systematic debugging with 6 phases: feedback loop → reproduce → hypothesise → instrument → fix → cleanup.
- **`/tdd`** — test-first workflow with red-green-refactor loop and vertical-slice discipline.
- **`/zoom-out`** — strategic re-grounding when you're lost in the weeds.

## Skill override paths

Any user can replace an embedded skill body without recompiling:

```
<workdir>/.archon/skills/<name>.md       (flat-file project, highest)
<workdir>/.archon/skills/<name>/SKILL.md (subdir project)
~/.config/archon/skills/<name>.md        (flat-file user)
~/.config/archon/skills/<name>/SKILL.md  (subdir user)
embedded fallback                        (binary, lowest)
```
