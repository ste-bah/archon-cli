# v1.2.8

v1.2.8 closes the auto-compaction PRD gaps left after the resume-state fixes.

## Auto-Compaction

- Auto/reactive compaction now uses provider-backed summary generation instead
  of synthetic summary text.
- Structured `CompactionOutcome` keeps `after_estimated_tokens` while accepting
  legacy `after_tokens` telemetry on read.
- `CompactionError` distinguishes provider failure, cancellation, invalid
  summary, and no-safe-boundary outcomes so the circuit breaker can count only
  real failures.
- Main sessions and subagents use the shared `UsageAccumulator` and retry
  context-window overflow once for initial and mid-stream failures.

## Context Windows

- `[context]` now exposes `context_window_override`, `output_reserve_tokens`,
  and `preflight_safety_margin`.
- The bundled/user `context.toml` catalog is used before provider metadata, and
  source labels are shown in `/context` and the TUI.
- Claude Sonnet 4.6 and Opus 4.7 paid-plan entries are set to 1M tokens; Codex
  auth GPT entries remain configurable in the same catalog.
- A guard script blocks new raw `200_000`/`200000` context-window literals in
  production Rust code.

## Pipeline And TUI

- Pipeline context handling now lives in facade/runner prompt budgeting, not the
  LLM adapter. The adapter no longer keeps cross-agent compaction state.
- `PromptBudget` applies context window, output reserve, preflight margin, and a
  tighter retry budget after context overflow.
- The TUI status bar tracks context name, source, and warning state. Warning is
  yellow bold; critical is red bold.
- `Ctrl+O` opens foreground parent/subagent activity output; `Ctrl+B` returns it
  to the background without cancelling the actor.

## Prompt Cache

- Prompt cache behavior is controlled by `prompt_cache`, `prompt_cache_mode`,
  `prompt_cache_ttl`, and `prompt_cache_conversation`.
- Anthropic-shape cache hints are stripped for unsupported providers and capped
  at four explicit breakpoints. One-hour TTL is emitted only when configured.
- `/context` now reports cache creation/read token totals.
