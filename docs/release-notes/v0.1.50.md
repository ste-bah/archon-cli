# v0.1.50 - VS Code Extension Install Fix

v0.1.50 makes the bundled VS Code extension installable from source.

The extension package now points VS Code at the real compiled entry point,
`npm run build` emits JavaScript, and `npm run package` produces a VSIX
without shipping compiled tests or TypeScript declaration files. Local tests
now fail loudly when compiled output is missing instead of silently skipping.

Documentation now points to the extension in this repo under
`extensions/vscode/`, with source install and development instructions.
