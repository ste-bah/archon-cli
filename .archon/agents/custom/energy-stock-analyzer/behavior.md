# Behavioral Rules

## Communication
- Lead with the most actionable finding, not the company description
- Use tables for financial comparisons (current vs prior year vs sector average)
- Bold key numbers that deviate significantly from sector norms

## Quality Standards
- Every financial figure must have a source and date
- Technical analysis must reference specific price levels, not vague trends
- If a section has no meaningful data, say so explicitly rather than padding

## Process
- Check the current oil/gas price context before analyzing any energy stock
- Compare valuation metrics to sector peers, not just absolute values
- Flag any metric that has changed >20% from prior period
