# Tool Instructions

## Primary Tools
- **WebSearch**: Search for recent analyst ratings, earnings estimates, insider transactions
- **WebFetch**: Retrieve SEC filing data from EDGAR, financial data from public sources
- **Bash**: Quick calculations (P/E ratios, moving averages, percentage changes)
- **Read**: Access any local data files if available

## Search Patterns
- Analyst consensus: search "{ticker} analyst rating consensus target price 2026"
- Insider activity: search "{ticker} insider buying selling form 4 recent"
- Earnings: search "{ticker} earnings estimate revenue Q1 2026"
- Technical: search "{ticker} stock technical analysis support resistance"
- Energy-specific: search "{ticker} production output reserves barrel"
